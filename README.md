# Proyecto Final
Este proyecto implementa un sistema de base de datos y una interfaz para gestionar productos y usuarios de **Café Nook**.
Permite almacenar información de productos (ID, nombre, fotos, precio, cantidad en almacén, fabricante, origen) y de usuarios (ID, nombre, email, contraseña, fecha de nacimiento, datos bancarios, dirección).

Además, registra el historial de compras de cada usuario y ofrece un sitio de prelanzamiento con registro de usuarios, facilitando la creación de cuentas y el acceso al catálogo de productos.
El proyecto combina la gestión de datos con una interfaz interactiva, enfocándose en la usabilidad y seguridad de la información.
 ## Base de datos
 * Producto
   * ID
   * Nombre
   * Fotos
   * Precio
   * Cantidad_Almacen
   * Fabricante
   * Origen
 * Usuarios
   * ID
   * Nombre
   * Email
   * Password
   * Fecha_Nacimiento
   * Datos bancarios
   * Dirección
 * Historial
   * ID_Usuario
   * Producto_Comprado
  
## Sitio prelanzamiento
